CSP-J/S 简介

CSP-J/S 是由 CCF（中国计算机学会，China Computer Federation）举办的非专业级别软件能力认证

认证分为 **CSP-J（入门级，junior）**和 **CSP-S（提高级，senoir）**两个组别，任何人都可以报名

认证形式：分为第一轮（初赛）和第二轮（复赛）两个认证阶段，第一轮成绩优异者（达到省/市分数线）才能进入第二轮，初赛一般是笔试（也有机试），第二轮是机试

认证者可以同时参加 CSP-J 和 CSP-S

官网：www.noi.cn （可以查看大纲、比赛时间等等）

第一轮试卷形式：

只包含选择题和判断题，机器阅卷（答题卡），满分100分

第一部分：单项选择题，共15题，每题2分，一共30分，考察 计算机基础知识、算法和数据结构

第二部分：阅读程序题，一般是三个程序，每个程序下有3-4题判断题和2-3题单选题，一般判断题1.5分一题，单选题3分一题，一共40分，考察程序阅读能力和代入计算能力

第三部分：完善程序题，一般是两个程序，每个程序下有5道单选题，每题3分，一共30分，考察程序阅读和编写能力

题型大概长什么样？

第一部分的单项选择一般都是计算机常识，选一个正确的

第二部分的阅读程序题每个题给一个程序，判断题给一句话问你对不对，选择题一般会问输入是……，输出是什么？

第三部分的完善程序题跟平常做题比较像，首先给一个题面，然后给你一个残缺的程序，让你选择一个语句填空

注意事项：

不要迟到 15分钟后不能进入

不能早退 必须做够2小时，除非身体不舒服

不能带与考试无关的东西 只能带笔、橡皮、无存储功能的手表，草稿纸考场会发，如果想跟家长联系，手机也必须关机放在考场外面

什么东西都不要带出去

不会做就蒙（懂的都懂）

不要气馁（自己不会别人也不会）

有问题举手问监考老师

计算机发展史与基本结构

计算机发展的重要人物

约翰·冯·诺依曼（John von Neumann，1903年12月3日-1957年2月8日），二十世纪著名美籍匈牙利 数学家、计算机科学家、物理学家

冯·诺依曼早期在研究算子理论、共振论、量子理论、集合论，开创了冯·诺依曼。曾参与曼哈顿计划，为第一颗原子弹的研制做出了贡献

被称为 现代计算机之父（冯·诺依曼体系结构）、博弈论之父 （人的竞争与合作的理论）

艾伦·麦席森·图灵（英语：Alan Mathison Turing，1912年6月23日~1954年6月7日），英国数学家、逻辑学家

被称为 计算机科学之父、人工智能之父

提出了一种用于判定机器是否具有人类智能的实验方法，图灵试验。图灵提出的 图灵机模型 为现代计算机的逻辑工作奠定了基础

为了纪念图灵，美国计算机协会（ACM）在 1966年 设立了 图灵奖，用来表彰在计算机科学领域做出巨大贡献的科学家，是计算机领域的最高奖项

注意：没有诺贝尔计算机奖，因为在开始颁发诺贝尔奖的时候还没有计算机

姚期智（清华姚班创立者）是唯一获得图灵奖的中国人

世界上第一台计算机：1946年2月14日（情人节），**ENIAC（Electronic Numerical Integrator And Computer）**诞生于 美国宾夕法尼亚大学。长 30.48 米，宽 6 米，占地面积约 170 平方米，重达 30 英吨，计算速度：每秒 5000 次加法或 400 次乘法

建造这台计算机的目的：美国和德国当时在打仗，然后会发射导弹，但总是发射不准，于是发明了计算机来计算导弹轨迹

冯·诺依曼提出了计算机制造的三个基本原则，采用 二进制逻辑（莱布尼茨发明），程序存储执行，计算机由五个部分组成（运算器、控制器、存储器、输入设备、输出设备），被称作冯·诺依曼体系结构

二进制逻辑：计算机背后由二进制实现

程序存储执行：把需要的程序先存储下来，用到的时候再执行

五大部分：

运算器：用来计算；控制器：控制整个流程；存储器：存储数据；输入设备：把数据输入到计算机里面；输出设备：将结果输出

流程：控制器控制着输入设备、输出设备和存储器，运算器和控制器一起放在CPU里面（计算机的大脑），输入设备读取了一些数据后放在存储器里面，你想算什么就交给运算器，运算器算完后再把结果放进去，然后将需要的东西输出

ENIAC 其实并不符合冯·诺依曼体系结构，冯·诺依曼提出这个结构的时间点在 ENIAC 诞生的时间之后

第一个给计算机写程序的人：阿达·洛芙莱斯（Ada Lovelace，1815年12月10日-1852年11月27日），著名英国诗人拜伦的女儿

为什么计算机的发明时间是 1946 年，而第一个给计算机写程序的人死于 1852年11月27日，时间对不上？

其实阿达·洛芙莱斯写的并不是计算机的程序，而是分析机的

故事简介：

在1834年，阿达的朋友——英国数学家、发明家兼机械工程师查尔斯·巴贝其（Charles Babbage）——发明了一台分析机；阿达则致力于为该分析机编写算法。在1842年，人称“数字女王”的阿达·洛芙莱斯（Ada Lovelace）编写了历史上首款电脑程序。并于1843 年公布了世界上第一套算法。

什么是分析机？分析机是由英国数学家查尔斯·巴贝奇设计的一种机械式通用计算机。分析机由蒸汽机驱动，大约有30米长、10米宽。它使用打孔纸带输入，采取最普通的十进制计数。

为什么被称作第一个给计算机写程序的人？巴贝的分析机后来被认为是最早期的计算机雏形，而阿达的算法则被认为是最早的计算机程序和软件。

为了纪念阿达，在1980年12月10日，美国国防部制作了一个新的高级计算机编程语言——Ada

计算机的基本结构

下图为基本结构的梳理图，接下来会一一介绍图中的内容

CPU

中央处理器 CPU（Center Process Unit） 是系统的 运算和控制核心 ，是信息处理、程序运行的最终执行单元。包含 控制单元（控制器）、算术逻辑单元（运算器）、存储单元（寄存器)

计算机 CPU 的 字长 从 4 位到 8 位、16 位、32 位处理器发展到了现在的 64 位

位（bit）在二进制里面只占一位（一个 0 或一个 1），一般描述文件大小都是以 字节（Byte） 描述，1 Byte = 8 bit，那么 64 位字长的 CPU 一次就可以处理 8 个字节，像软件的大小都是 ……B，这个 B 就是字节的意思，例如 KB 就是千字节

CPU 的主频 指 CPU 的工作时钟频率，一般来说，主频越高，完成任务的速度也越快

GPU

图形处理器 GPU（Graphics Processing Unit） 是用来渲染图像的，一般在人工智能方面用的比较多，玩游戏也需要这个东西。不过要注意的是 GPU 并不等于人们常说的显卡，GPU是显卡的核心，显卡则是由GPU、显存、电路板，还有BIOS固件组成的。

存储设备

存储器分为 Cache（缓存）、内存（主存）、外存（辅存），用来存储程序和数据

外存：外存就是在外部的存储设备**（唯有硬盘是例外，硬盘放在机箱里面）**，例如 硬盘、U 盘、光盘等

内存：内存分为 RAM（Random Access Memory）随机存取存储器 和 ROM（Read-Only Memory）只读存储器

在内存中，每个存储单元都有 唯一 的编号，被称为 地址。好比人们的住宅，房子就是存储单元，而门牌号就是地址

缓存：Cache（缓存，高速缓冲存储器），是位于 CPU 与内存间 的一块 容量较小但速度很快的存储器

下面介绍一下各个存储之间的速度：

上文介绍 CPU 时已经提到，CPU 里面有寄存器，因为所有的东西最后都是放在 CPU 里面算的，那么离得远近访问越容易，速度也就快，所以说寄存器的计算速度最快，不过寄存器能存的东西非常少，缓存是除了寄存器离 CPU 最近的，内存在缓存的外面，最后是外存，外存最慢

总结：离 CPU 越近就越快，距离顺序为 外存 > 内存 > 缓存 > CPU 里面的寄存器

存储单位的换算：

注意：小 b 是 bit，大 B 是 Byte

1 B = 8 b 1 KB = 1024 B 1 MB = 1024 KB 1 GB = 1024 MB 1 TB = 1024 GB 1 PB = 1024 TB

为什么是 1024？ 因为 1024 跟 1000 最接近，也是 2102^{10}210

常见的内存容量：1 GB，4 GB，8 GB，16 GB

常见的硬盘容量：256 GB，512 GB，1 T

总结：离 CPU 越远越慢，而且容量也越大

计算机常用软件和语言

计算机软件系统

计算机软件系统是指 计算机在运行的程序各种程序、数据及相关的文档资料。计算机软件系统通常分为 系统软件 和 应用软件

系统软件：负担控制和协调计算机及其外部设备、支持应用软件的开发和运行的一类计算机软件。系统软件一般包括 操作系统、语言处理系统、数据库系统、网络管理系统 等

应用软件：为特定领域开发、并为特定目的服务的一类软件，是直接面向用户需求的，帮助用户提高工作效率和质量，解决一些问题

计算机语言

计算机语言（计算机说的话）：机器语言、汇编语言、高级语言

机器语言 就是机器直接能看懂的语言，使用 二进制代码

汇编语言 是在机器语言的缩写上，用一些单词代替特别的指令 比如 ADD 代表 加法操作，然后再用一个翻译器翻译成机器语言，这样机器就看懂了

汇编语言和机器语言都称为 低级语言，依赖于硬件，怎么写可能依赖于 CPU 的型号和各种设备，可读性和可移植性差。

高级语言编写的程序独立于，方便移植，但是必须用个翻译器翻译成机器语言才能执行

在高级语言中，有两种翻译方法，一种是 解释，一种是 编译，每种高级语言翻译的方法不一样，根据翻译方法的不同，高级语言又分为 解释性语言 和 编译性语言

C++ 就是编译性语言，编译实际上就是把 .cpp 翻译成 .exe（可执行程序），对于编译性语言，首次运行需要编译，再次运行就不需要了，直接用之前编译的结果。常见的编译性语言有：C/C++、Pascal、Delphi、Rust、Go 等

Java 就是解释性语言，和编译性语言不同，运行一次就要翻译一次，所以效率一般比较低。常见的解释性语言有：Java、C#、PHP、JavaScript、Python 等

编译性语言的优缺点：

优点：运行速度快，代码效率高，编译后的程序不可修改，保密性较好。

缺点：代码需要经过编译方可运行，可移植性差，只能在兼容的操作系统上运行 。

解释性语言的优缺点：

优点：可移植性较好，只要有解释环境，可在不同的操作系统上运行。

缺点：运行需要解释环境，运行起来比编译的要慢，占用资源也要多一些，代码效率低，代码修改后就可运行，不需要编译过程。

高级语言还有另外一种分类的角度，从解决问题的角度可以分为 面向对象语言 和 面向过程语言

面向过程语言也称为结构化程序设计语言，问题被看作一系列需要完成的任务，函数则用于完成这些任务，解决问题的焦点集中于 函数。采用 自顶向下、逐步求精的程序设计方法

常见的面向过程语言：Fortran、Pascal、C 等

面向对象语言是一类以对象做为基本程序设计结构的程序设计语言，指用于描述的设计是以 对象 为核心，而对象是程序运行时刻的基本成分

常见的面向对象语言：C++、C#、Java、Python、JavaScript、Rust、Go

第一个高级语言 Fortran

面向对象语言的三个基本特征：封装、继承、多态

前缀、中缀、后缀表达式

中缀表达式是最常用的算术表示方法

但是这种方法需要分析括号优先级等等，对于计算机不友好，所以数学家发明了一种 不含括号 的表达式描述方法

前缀表达式，也称 波兰式，运算符在前，操作数在，实际上就是上文中表达式树的 先序遍历

后缀表达式，也称 逆波兰式，运算符在后，操作数在前，实际上就是上文中表达式树的 后序遍历

后缀表达式的计算方法（计算机最容易看懂的一个）：

  1. 从左至右扫描表达式
  2. 遇到数字时，将数字压入堆栈，遇到运算符时，弹出栈顶的两个数，用运算符对它们做相应的计算（次顶元素op 栈顶元素 ），并将结果入栈
  3. 重复上述过程直到表达式最右端，最后运算得出的值即为表达式的结果

前缀表达式只需要将第一步改成 从右至左扫描表达式就行了
