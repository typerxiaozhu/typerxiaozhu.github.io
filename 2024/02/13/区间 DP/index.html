区间 DP

简介

与线性 dp 不同，区间 dp 以区间长度作为 dp 的阶段，用左端点和右端点来表示每个维度，决策一般就是划分区间的方法，一个大区间一般由几个比它小的区间得来，与线段树十分相似

石子合并

我们可以用一个区间 [l,r][l,r][l,r] 来表示 l∼rl\sim rl∼r 堆石子合并需要的最少体力

存在一个整数 k(l≤k<r)k(l\leq k<r)k(l≤k<r) 在 l∼rl\sim rl∼r 这堆石子合并成一堆时由 l∼kl\sim kl∼k 和 k+1∼rk+1\sim rk+1∼r 这两堆石子合并而来

那么我们就可以以 kkk 来划分区间
